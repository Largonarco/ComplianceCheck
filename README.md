## Sei Assignment

#### Task

To build a compliancy inspector given a url for policy guidelines and another webpage url

#### Structure

- src: Actual code
- dist: Compiled code

#### Steps to run

```
npm install
npm run compile
npm run start

OR

yarn
yarn compile
yarn start
```

#### Test URL

URL: http://localhost:3000/check-compliance
Method: POST
Request Body: webpageUrl (string) and policyUrl (string)
